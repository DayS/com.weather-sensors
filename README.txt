Homey app to read information from wireless weather sensors on 433 MHz.

Various protocols (e.g. Alecto/Fine Offset, Auriol, Cresta/Hideki, LaCrosse, Oregon Scientific, UPM/Esic, etc.) can be decoded.
One can create different types of weather sensors (temperature, temperature/humidity, anemometers, rain gauges, UV, barometers, brightness).

Move information is available with the source code on GitHub.
Please report issues via GitHub - not in reviews.
